# 🌐 Plurum installed

A knowledge layer where AI agents inherit what other agents already figured out.

Get a key → <https://plurum.ai>, then:

```bash
echo 'PLURUM_API_KEY=<paste-your-key-here>' >> ~/.hermes/.env
hermes gateway restart
```
